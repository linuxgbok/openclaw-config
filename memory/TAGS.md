# 记忆分类标签系统

## 标签规范

使用 `#标签` 格式在记忆文件中标记内容类型：

### 标签列表

| 标签 | 用途 | 示例 |
|------|------|------|
| `#决策` | 重要决策记录 | 选择飞书作为主要渠道 #决策 |
| `#教训` | 踩过的坑 | 创建文档后必须验证 #教训 |
| `#项目` | 项目相关 | TaskReminder iOS 项目 #项目 |
| `#配置` | 配置变更 | 优化 Token 消耗 #配置 |
| `#偏好` | 用户偏好 | 偏好 Python 和 JavaScript #偏好 |
| `#待办` | 待办事项 | 完善记忆系统 #待办 |

## 使用示例

```markdown
## 今日事件
- ✅ 完成飞书配置 #配置 #决策
- ❌ 发送空白文档给用户 #教训
- 📝 TaskReminder 代码审查 #项目
```

## 搜索技巧

按标签搜索记忆：
```bash
grep -r "#决策" ~/.openclaw/workspace/memory/
grep -r "#教训" ~/.openclaw/workspace/MEMORY.md
```

## 自动分类规则

- 包含 `配置`、`设置` → 自动标记 #配置
- 包含 `错误`、`失败`、`问题` → 自动标记 #教训
- 包含 `选择`、`决定`、`使用` → 自动标记 #决策
